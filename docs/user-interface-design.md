# UI/UX Design Specification

## Design Philosophy & Appearance
- **Design Goal:** Minamilst polished asthetic to focus users attention on exercises.
- **Colors:** Soft neutral colors with high-contrast accent colors for important interactive elements.
- **Layout:** Vertical layout focusing on fitting in a single page of the limited space provided by mobile displays.
