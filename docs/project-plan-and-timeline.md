 # Project Plan and Timeline


## Project Goals


#### Develop User Interfaces
Create responsive, mobile-first frontend screens with functional UI elements
#### Implement User Account Management
Build secure user login, registration, and session management systems
#### Design a Database Schema
Plan the relational tables required to effectively map and connect user accounts, lesson content, quiz questions, flashcards, and progress tracking.
#### Implement Database Schema
Execute the planned schema by creating the MySQL database and connecting the front to read, write, and update the database.
#### Refine Kana Exercise Experience
Polish the interactive Hiragana and Katakana exercises, ensuring smooth user experience, and a wide variety of exercises.
#### Add Instructive Information To UI
Integrate explanatory content into the interface, such as the "Explain Kana" and "Explain Listening Exercises" screens, to guide beginners and lower the barrier to entry.
#### Implement Storage of User Settings
Develop the backend logic required to securely save, retrieve, and update custom user preferences and profile information in the database.
#### Ensure User Account Deletion Works
Create a secure, reliable mechanism for users to permanently delete their accounts and associated progress data.
#### Implement Security Layer
Apply password encryption, secure session handling, and other security methods to ensure the security of user information and the application.
#### Develop Kana Home
Build the central hub where users can view their current Kana mastery progress.
#### Create and Refine Listening Exercises
Develop audio-based exercises, ensuring they are well explained to the user that the expereince is smooth.
#### Implement Storage of User Learning Progress
Store user's learning progress in the database and update it as they complete lessons. Utalize this information to provide the best lessons to users.
#### Develop Software Testing Plan
Draft a structured approach to verify all functional and non-functional requirements.
#### Develop Listening Home
Construct the dashboard for users to monitor their listening exercise progress and easily select their desired level of audio training.
#### Perform User Acceptance Testing
Conduct hands-on testing with stakeholders or representative users to confirm the application successfully serves as a low-barrier entry point for Japanese language learners.
#### Perform Quality Testing and Ensure Polish
Execute final performance, scalability, and bug testing across the LAMP stack to guarantee a smooth and polished user experience.
#### Create End-User Documentation
Write clear, accessible instructions and guides to help new users navigate the application's flow and understand the underlying learning methodology.
#### Finalize Project and Documentation
Complete all developer-facing documentation, update the GitHub repository.
#### Final Report
Create a report of delivery to client and results of the project.

## Project Timeline

The project will span an additional 9-week period from March 1st to May 10th.
